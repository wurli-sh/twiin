# @twiin/shared — TypeScript Shared Library

Single source of truth for ABIs, deployed addresses, chain constants, digest helpers, and 6551 derivation helpers. Consumed by `apps/backend` and `apps/frontend`.

**Status: Phase 2 — not yet implemented.**

Pre-conditions before starting Phase 2:

- `packages/contracts` compiles (`pnpm compile` in contracts)
- Local deploy works (`pnpm deploy:local` writes `addresses.json`)
- Somnia deploy produced `packages/shared/addresses.json`

## Planned File Layout

```
packages/shared/
├── abis/                    — generated from hardhat artifacts (do not edit by hand)
│   ├── TwiinFactory.json
│   ├── TwiinAgent.json
│   ├── TwiinAccount.json
│   ├── TwiinNames.json
│   ├── AgentRegistry.json
│   ├── AgentVault.json
│   ├── AgentPolicy.json
│   ├── AgentOrchestrator.json
│   └── OracleFeed.json
├── addresses.json           — auto-generated by contracts/scripts/deploy.ts
├── constants.ts             — CHAIN_ID, TWIIN_6551_SALT, native configIds 0-5, capability IDs, Step/Task enum mirrors
├── digest.ts                — buildTwiinDigest() matching AgentOrchestrator.submitExternalResult
├── twiin-account.ts         — deterministic 6551 account address derivation helper
└── index.ts                 — single import surface (re-exports everything)
```

## Key Exports to Implement

### `constants.ts`

```ts
export const CHAIN_ID = 50312;
export const TWIIN_6551_SALT = "0x" + "00".repeat(32);  // bytes32(0)
export const NATIVE_CONFIG_IDS = { JANICE: 0, WEB_INTEL: 1, ORACLE: 2, ANALYSIS: 3, REPORTER: 4, EXECUTOR: 5 };
export const CAP_IDS = { PLAN_TRUSTLESS: ..., WEB_SCRAPE: ..., JSON_FETCH: ..., LLM_ANALYZE: ..., LLM_REPORT: ..., ONCHAIN_EXECUTE: ... };
// Step/Task enums mirroring TwiinTypes.sol
```

### `digest.ts`

ECDSA digest must match exactly:

```
keccak256(abi.encodePacked("\x19Twiin External Result v1\n", chainId, orchestrator, taskId, stepIdx, externalRequestId, keccak256(result)))
```

then `toEthSignedMessageHash`.

### `twiin-account.ts`

Deterministic 6551 address derivation using the local `ERC6551Registry` bytecode (not the canonical registry absent on Somnia testnet). Address at `addresses.json → registry6551`.

## Conventions

- Runtime: Node >=20
- Dependencies: `viem` (EVM interactions), `zod` (validation)
- Pure TypeScript — no bundler required; consumer transpiles
- All ABIs copied from `packages/contracts/artifacts/src/**/*.json` by a build script
- `addresses.json` is written by `packages/contracts/scripts/deploy.ts` — never edit by hand
