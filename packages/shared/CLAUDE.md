# @twiin/shared ‚Äî TypeScript Shared Library

Single source of truth for ABIs, deployed addresses, chain constants, ECDSA digest helper, and ERC-6551 address derivation. Consumed by `apps/backend` and `apps/frontend`.

**Status: Phase 2 complete ‚Äî 22/22 vitest tests passing.**

## Commands

| Command           | Description                                                                                |
| ----------------- | ------------------------------------------------------------------------------------------ |
| `pnpm copy-abis`  | Extract ABIs from Hardhat artifacts ‚Üí `abis/` (requires `pnpm compile` in contracts first) |
| `pnpm test`       | Run vitest parity tests                                                                    |
| `pnpm test:watch` | Vitest watch mode                                                                          |

## Key Exports

### `constants.ts`

- `CHAIN_ID = 50312`
- `TWIIN_6551_SALT` = viem `zeroHash` (bytes32(0))
- `NativeConfigId` ‚Äî `{ JANICE:0, WEB_INTEL:1, ORACLE:2, ANALYSIS:3, REPORTER:4, EXECUTOR:5 }`
- `CapabilityId` ‚Äî 9 entries computed at runtime via `keccak256(toBytes(name))`; matches `deploy.ts` exactly
- Plain enums mirroring `TwiinTypes.sol`: `TaskState`, `StepState`, `AgentLane`, `PlanMode`
- `DEFAULT_DAILY_CAP_WEI = parseEther("2")`, `DEFAULT_MAX_PER_TASK_WEI = parseEther("1")`

### `digest.ts` ‚Äî `buildTwiinDigest(params): Hex`

Returns the **inner keccak256** (pre-EIP-191) matching `AgentOrchestrator.submitExternalResult`:

```
keccak256(encodePacked("\x19Twiin External Result v1\n", chainId, orchestrator, taskId, stepIdx, externalRequestId, keccak256(result)))
```

Sign with: `walletClient.signMessage({ account, message: { raw: buildTwiinDigest(p) } })`

### `twiin-account.ts` ‚Äî `deriveTwiinAccountAddress(params): Address`

Deterministic ERC-6551 CREATE2 derivation:

- CREATE2 salt = raw `TWIIN_6551_SALT` (NOT a hash of the footer)
- Footer = `encodeAbiParameters([bytes32, uint256, address, uint256], [salt, chainId, twiinAgent, tokenId])` (abi.encode, 128 bytes)
- initCode = `concat([PROXY_HEADER(20), impl(20), PROXY_TRAILER(15), footer(128)])`

### `index.ts` ‚Äî `loadAddresses(raw: unknown): Addresses`

Zod-validates and types the imported `addresses.json`. Fails loudly on stub. Pass imported JSON as argument ‚Äî works in both Node and browser (no `require` at import time).

## Conventions

- viem `^2.47.4` for all EVM primitives ‚Äî no ethers
- CommonJS module (`"module": "CommonJS"` in tsconfig) ‚Äî consistent with contracts package
- `addresses.json` is written by `contracts/scripts/deploy.ts` ‚Äî never edit by hand
- ABIs in `abis/` are generated by `pnpm copy-abis` ‚Äî never edit by hand
- `privateKeyToAccount` is in `viem/accounts`, not `viem`
